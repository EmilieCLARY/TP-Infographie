# TP02 - Courbes de Bézier avec three.js

## Information

Les fichiers `ap1.html`, `ap2.html`, `ap3.html`, `ap1.js`, ap2.js`, ap3.js` sont des copies de `courbesBezier.html` et `courbesBezier.js`.
La principale différence réside dans le tableau de points (correspondant aux trois applications) préenregistré dedans.

`courbesBezier.html` et `courbesBezier.js` sont considérés comme les **fichiers principaux** et seront **commentés**.

Afin de naviguer plus facilement à travers les fichiers, `main.html` vous propose un menu simple mais efficace pour naviger aisément. On vous recommande d'accéder aux diverses applications via `main.html`.

Les méthodes de traçage de courbe Bernstein et Casteljau sont activés simultanément. On affiche les polynômes de Bernstein (en violet) et la courbe via la méthode de Casteljau.

## Fichiers

- `berstein.js` : Méthode de Bernstein
- `casteljau.js` : Méthode de Casteljau
- `courbesBezier.html` : Page HTML du mode "libre"
- `courbesBezier.js` : Point d'entrée de notre code.
- `main.html` : Navigateur de solution
- `materialsettings.js` : Fichier de configuration des différents "materials" utilisés dans le projet (sert notamment à définir les couleurs).
- `maths.js` : Contient ~~l'unique~~ les formules mathématiques nécessaires (explicitation du coefficient binomial).
- `three.js` : Source de ThreeJS
